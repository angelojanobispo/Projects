Excel Compilation Project

Overview

This project is an organized compilation of data that was previously scattered across multiple Excel files. To ensure accuracy and completeness, I manually checked each entry one by one and compiled them into a single updated version.

Privacy Notice

For confidentiality, **student numbers and names are intentionally masked (e.g., `*******`)**. This ensures that private information is not exposed while still demonstrating the structure and process of the dataset.

Purpose
- Consolidate scattered Excel lists into one master file.
- Organize data **year by year** and **semester by semester**.
- Verify every entry against the previous files to maintain accuracy.
- Provide a clean, reliable dataset for future reference.

Process
1. Gathering Files
   Collected all the scattered Excel files containing partial lists.

2. Verification
   Checked each entry individually against the original files to avoid missing or duplicating data.

3. Compilation  
   Organized the data chronologically (by year and semester) and merged it into one updated Excel file.

4. Final Review  
   Ensured that the compiled version reflects the same information as the previous files, but in a structured and accessible format.

Output
- A single Excel file containing all verified records.
- Sensitive identifiers masked for privacy.
- Data arranged systematically for easier navigation and analysis.

Benefits
- Eliminates confusion caused by scattered files.
- Provides a reliable reference point for future updates.
- Saves time by centralizing all information in one place.
